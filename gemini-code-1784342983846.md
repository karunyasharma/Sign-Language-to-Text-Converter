# Sign Language to Text Converter

[cite_start]**Author:** Karunya Sharma [cite: 2]

## Introduction
[cite_start]Millions of people use sign language to communicate, but a significant barrier exists for non-signers to understand it[cite: 4, 5]. 
[cite_start]The objective of this project is to convert sign language gestures into readable text in real-time using a webcam and AI[cite: 6].

## How It Works
1. [cite_start]**Capture:** The webcam captures hand signs[cite: 8].
2. [cite_start]**Preprocessing:** Image processing is used to isolate hand gestures[cite: 9].
3. [cite_start]**Recognition:** A machine learning model classifies the hand signs[cite: 10].
4. [cite_start]**Translation:** Recognized signs are converted into corresponding text[cite: 11].
5. [cite_start]**Display:** Text output is displayed to the user[cite: 12].

## Setup Instructions
1. Clone this repository.
2. Train a model using [Google Teachable Machine](https://teachablemachine.withgoogle.com/) and export it as an H5 Keras model.
3. Place `keras_model.h5` and `labels.txt` into the `/models/` directory.
4. Upload `Sign_Language_Colab.ipynb` to Google Colab and run all cells.
